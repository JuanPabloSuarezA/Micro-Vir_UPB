# Micro-Vir_UPB


## Overview
Network-attached storage system for small business or home 

## Dependencies

+ MongoDB
+ Node.js
+ Json Web Token
+ Express.js
+ React.js
+ Nessus
+ jsonwebtoken
+ mongoose
+ Kali Linux
+ Owasp Zap
+ passport
+ passport-jwt
+ ibm-cos-sdk-js
+ GNU Privacy Guard


## Running this project

From the repo:  
1. Clone this project locally
2. Run `npm i` or `npm install` in the root of the project
3. Create `.env` and `.env.dev` file in the root of the project
4. Run `npm run tsc` in command line or bash
5. Run `npm run dev` in command line or bash

## Development usage

~~~
npm run tsc
npm run dev
~~~

## Production usage

~~~
npm start
~~~
